Blink
=====

About
-----

Turn a LED on and off periodically with a one second interval.

Source code
-----------

.. include:: blink/source-code.rst

The source code can also be found on Github in the
:github-tree:`examples/blink` folder.

Build and run
-------------

Build and upload the application.

.. code-block:: text

   $ cd examples/blink
   $ make -s BOARD=<board> upload
