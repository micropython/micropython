.. _uefi_tutorial:

MicroPython tutorial for the UEFI port
======================================

This tutorial gets you started running MicroPython in the UEFI firmware
environment -- under QEMU with OVMF/AAVMF firmware, or from the UEFI shell on a
real machine.  If it is your first time it is recommended to follow the sections
in order; otherwise they are mostly self-contained.

The tutorial assumes some familiarity with Python.  For an introduction to the
language itself see `<https://www.python.org>`__.
